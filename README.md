# User Retention & Engagement Analysis
### Author: Alisha Tajin

### Project Overview
This project simulates and analyzes user retention for a streaming service. It combines **SQL, Python, and machine learning** to explore how user demographics and subscription details influence churn.

#### Key Objectives:
- Generate simulated user and subscription data.
- Store and query data using SQLite.
- Predict churn using Logistic Regression.
- Evaluate model performance with accuracy, precision, recall, and F1-score.

#### Data:
- Users: age, country, signup date
- Subscriptions: subscription tier, monthly fee, churn status
- Simulated for 1,000 users with a 20% churn rate.

#### Skills & Tools:
- Python: pandas, numpy, matplotlib, scikit-learn
- SQL: SQLite for relational data management
- Machine Learning: Logistic Regression for binary classification
- Version Control: Git, GitHub
